SoundSource's Play-Thru Window

To access SoundSource's Play-Thru window, select “Open Play-Thru Window” from the global SoundSource menu. This window allows you to select an Input device from which to pull audio and an Output device through which that audio will be heard. You can also adjust the channels used for each device.

To hear audio from a specific input device, select it in the Input area. Next, set the Output device, then click the “Start Play-Thru” button. Any audio picked up by the input device will now be heard via the output. That's it!

Note: Be sure to avoid feedback loops, by isolating the output from the input. You can do this by sending a microphone's audio to headphones, so that the audio doesn't leak back to the microphone.
